## 2. CORE FRAMEWORKS (stable layer)

### Dual Decision Flow
All significant work moves through one of two parallel flows:

```
Risk        → Decision → Coordination → Deliverable
Opportunity → Decision → Coordination → Deliverable
```

Risk-driven flow: triggered by threat detection, misalignment, or project stress.
Opportunity-driven flow: triggered by value creation, scope expansion, or strategic positioning.
Both flows share the same execution spine. The trigger determines posture; the structure determines quality.

### Role Hierarchy
```
Principal = Decisions / Client Relationships / Risk
PM        = Coordination / Execution
APM       = Information / Tracking
TPM       = Redundancy / Continuity (Secondary PM layer)
```

### Professional Value Equation
```
Value = Clarity + Foresight + Alignment
NOT production volume or technical output
```

### AI Positioning
```
Alpine = AI-Augmented Decision Advisor
NOT administrative project manager
```

AI compresses production and analysis.
Human value concentrates in decision-making, coordination, and advisory roles.
